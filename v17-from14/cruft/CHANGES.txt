v8-opt7 CHANGES - Minimal libc Usage
=====================================

BINARY SIZE
-----------
v7-opt6:  23,408 bytes
v8-opt7:  15,552 bytes
Savings:  7,856 bytes (33.56% reduction)

BIGGEST WINS
------------
1. Removed stdio.h:        -8,000 bytes (printf/snprintf overhead)
2. Removed string literals: -3,101 bytes (.rodata section)
3. Removed PLT indirection: -1,520 bytes (dynamic linking)
4. Code size reduction:     -2,175 bytes (.text section)

MAKEFILE CHANGES
----------------
Added flags:
  -fno-builtin    # Don't use libc builtin functions
  -fno-plt        # No PLT indirection

SOURCE CODE CHANGES
-------------------
1. Removed: #include <stdio.h>
2. Removed: 99+ printf() calls
3. Removed: 1 snprintf() call
4. Replaced: Manual string construction instead of snprintf
